Pequeña aplicación para controlar el tiempo del parquímetro y advertir antes de que se venza. 
Solo hay que ingresar el tiempo que nos da el parquímetro (en minutos), 
Posteriormente hacer clic en el botón verde para guardar la ubicación cuando estamos todavía cerca de él.
La app nos informa que la ubicación y el tiempo han sido configurados.
Es entonces que empieza a correr el tiempo, una alarma sonará cuando falten 5 minutos para que se acabe el tiempo.
Cuando lo anterior suceda además de sonar la alarma, aparecerá un aviso con un botón rojo, al hacer clic en él se detendrá la alarma.
En caso de no recordar donde se dejó el auto, se debe hacer clic en el botón azul que se encuentra en la parte inferior y se abrirá Google maps.
Probablemente nos pida permiso para acceder a la ubicación y entonces nos mostrará la ruta a utilizar para llegar a donde dejamos el auto
Para volver a ingresar tiempo del parquímetro nuevamente, simplemente escribir el tiempo en recuedro de Tiempo parquímetro
